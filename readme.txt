Pour utiliser mon Programme veuillez suivre les étapes suivantes

+Entrez le choix l'opération à éffectuer (1 pour crypter un message et 2 pour décrypter un message)

*Dans le cas du cryptage : 
-Vous devez entrez le message à crypter 
-Ensuite entrez le décalage à appliquer qui doit être un entier strictement positif

*Dans le cas du décryptage : 
-Vous devez entrez le message à décrypter 
-Ensuite entrez le décalage à appliquer qui doit être un entier strictement positif
